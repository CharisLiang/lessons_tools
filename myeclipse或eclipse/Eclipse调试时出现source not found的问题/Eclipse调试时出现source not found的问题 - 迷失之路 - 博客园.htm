Eclipse调试时出现source not found的问题

Eclipse中新加载了一个java项目，打断点debug时，Debug小窗口显示source not found，网搜找到了解决办法，在此记录一下。

 

原文地址：http://blog.csdn.net/eoeandroida/article/details/41282783

原文作者：思益炫

 

解决方法：

1.  点击debug窗口中的'Edit Source Lookup Path'按钮。

2.  在弹出窗口中点击'Add'按钮。

3.  在弹出窗口中点击'Project'选项。

4.  选择自己正在调试的项目。

5.  不要勾选 'Search for duplicate source files on the path'这个选项。

6.  保存，重启eclipse。
